# Deployment with `kubectl` and `kind`

#### Install
```bash
brew install kubectl
brew install kind
```

#### Setup
```bash
kubectl apply -f deployment.yaml
kubectl get pods

kubectl apply -f service.yaml
kubectl get svc
```

#### Load docker image
```bash
kind load docker-image zoomcamp-model:3.13.10-hw10
```

#### Start service
```bash
kubectl port-forward service/subscription 9696:80 
```

#### Delete everything in one namespace
```bash
kubectl delete all -l app=subscription
```
